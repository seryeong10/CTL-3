# 배움페이
디지털 역량 향상형 지역화폐 기반 ESG 지역상생 플랫폼 
---------------------------------------

📌 서비스 소개

키오스크, 예약, 온라인 주문 등 실생활 디지털 기능을 시뮬레이션 형태로 학습하고,미션 완료 시 포인트를 지급받아 
지역 가맹점에서 사용할 수 있도록 연결하는 서비스
  
**디지털 학습 → 포인트 보상 → 지역 소비** 구조


🎯 핵심 문제

- 키오스크, 모바일 결제 등 디지털 서비스 증가
- 고령층 및 디지털 취약계층의 이용 어려움
- 기존 교육의 낮은 지속성 및 동기 부족
- 학습과 지역 소비 간 연결 부재


👥 주요 사용자

- 디지털 기기 사용이 어려운 시니어
- 디지털 학습이 필요한 중장년층
- 지역 상권 및 지자체 (간접 참여자)
  

⚙️ 주요 기능

### 1. 디지털 미션 학습
- 키오스크 / 예약 / 온라인 주문 시뮬레이션
- 난이도별 미션 제공
- 성공 시 포인트 지급

### 2. 사용자 맞춤 추천
- 사용자 수행 기록 기반 미션 추천
- 난이도 자동 조정 (쉬움/보통/어려움)

### 3. 포인트 시스템
- 미션 수행 시 포인트 적립
- 하루 최대 지급 제한
- 반복 학습 가능

### 4. QR / 바코드 결제
- 적립 포인트 지역 가맹점 사용 가능
- QR 또는 바코드 결제 방식 지원

### 5. 출석 및 반복 학습
- 출석 체크 기능
- 반복 학습 시 추가 보상 제공


🌱 기대 효과

- 디지털 역량 향상
- 학습 지속 동기 강화
- 지역 상권 활성화
- 세대 간 디지털 격차 완화
